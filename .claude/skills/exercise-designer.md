Design effective hands-on practice exercises that reinforce learning objectives and build practical skills.

WHAT IT DOES:
Creates structured exercises with clear instructions, starter code (if needed), success criteria, and expected outcomes.

EXERCISE COMPONENTS:
- Objective: What skill this exercise builds
- Instructions: Step-by-step what to do
- Starter code/template: Foundation to build on
- Success criteria: How to know it's correct
- Expected output: What should happen
- Extension challenges: For advanced learners

TYPES:
- Guided Practice: Step-by-step with hints
- Independent Practice: Apply concepts without guidance
- Challenge Problems: Integrate multiple concepts
- Debugging Exercises: Find and fix errors
- Design Tasks: Create from scratch

USE FOR: End-of-chapter exercises, lab activities, practice problems, skill-building tasks.
