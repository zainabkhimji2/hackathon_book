Create comprehensive assessments (quizzes, projects, rubrics) that accurately measure learning outcomes.

WHAT IT DOES:
Designs multiple-choice questions, short answer questions, coding challenges, and project rubrics aligned with learning objectives.

ASSESSMENT TYPES:
- Multiple Choice: Test conceptual understanding
- Short Answer: Test explanation ability
- Code Completion: Test syntax and logic
- Debugging: Test error identification
- Project Rubrics: Evaluate comprehensive work

QUESTION QUALITY:
- Align directly with learning objectives
- Test understanding, not memorization
- Include realistic scenarios
- Provide clear correct answers
- Add explanations for why answers are correct/incorrect

RUBRIC FORMAT:
Criteria | Excellent (4) | Good (3) | Adequate (2) | Needs Work (1)

USE FOR: Weekly quizzes, chapter assessments, project grading, capstone evaluation.
