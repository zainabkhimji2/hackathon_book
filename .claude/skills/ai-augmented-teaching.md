Design lessons that effectively integrate AI tools (chatbots, code assistants, personalized tutors) into the learning experience.

WHAT IT DOES:
Plans how AI features enhance learning without replacing human teaching. Designs chatbot interactions, personalized content paths, and AI-assisted exercises.

AI INTEGRATION TYPES:
- RAG Chatbot: Answer questions about content
- Code Assistant: Help debug and explain code
- Personalized Tutor: Adapt content to background
- Practice Generator: Create custom exercises
- Translation: Multi-language support

DESIGN PRINCIPLES:
- AI augments, doesn't replace learning
- Encourage critical thinking about AI responses
- Provide feedback on AI interactions
- Design for different skill levels
- Maintain human expert oversight

USE FOR: Designing chatbot features, planning personalization, creating AI-enhanced exercises.
