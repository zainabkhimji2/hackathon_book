# Week 4: ROS 2 Part 2

## Chapter Overview

Build on your ROS 2 foundation by learning advanced communication patterns, custom message types, and automated system launch configurations.

## Learning Objectives

By the end of this chapter, you will be able to:

- **Implement** ROS 2 services for request-reply communication
- **Create** ROS 2 actions for long-running tasks with feedback
- **Design** custom message types for your specific needs
- **Write** launch files to start multi-node systems
- **Build** a complete publisher-subscriber system

## Prerequisites

- Completion of Week 3: ROS 2 Part 1
- Understanding of ROS 2 nodes, topics, and basic pub-sub pattern
- Python class inheritance and async programming (helpful)
